# 📰 Django Form Article Project

A full-featured article management web application built with Django, supporting complete CRUD operations with user authentication.

---

> Coming soon via deployment

## ✨ Features

- 📝 **Create Articles** — Write and publish new articles via Django Forms
- 📖 **Read Articles** — Browse and view all published articles
- ✏️ **Update Articles** — Edit existing articles with ease
- 🗑️ **Delete Articles** — Remove articles securely
- 🔐 **User Authentication** — Login & signup to manage your own articles

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/keerthana22r-oss/django_form.git
   cd django_form
   ```

2. **Create a virtual environment**
   ```bash
   python -m venv myenv
   myenv\Scripts\activate   # Windows
   ```

3. **Install dependencies**
   ```bash
   pip install django
   ```

4. **Run migrations**
   ```bash
   python manage.py migrate
   ```

5. **Start the server**
   ```bash
   python manage.py runserver
   ```

6. **Open in browser**
   ```
   http://127.0.0.1:8000
   ```
